# Python Programming Foundations & Logic Projects

## 📌 Project Overview
This repository contains a collection of Python projects and assignments demonstrating core programming concepts. It ranges from basic string manipulations to a full Capstone project focused on automated data cleaning and insight generation.

## 🛠️ Core Competencies
*   **Basics:** User input/output, type casting, and arithmetic operations.
*   **Data Structures:** Deep dives into Lists, Tuples, Sets, and Dictionaries.
*   **Control Flow:** Implementation of `if-elif-else` statements, `for` loops for iteration, and `while` loops for conditional logic.
*   **Text Processing:** String slicing, indexing, and advanced cleaning methods.
*   **Project Logic:** Building functional applications like multiplication generators, BMI calculators, and feedback analysis tools.

## 📂 Project Portfolio

### 1. Capstone Project: Automated Feedback Analysis
A functional script designed to process and analyze customer feedback data.
*   **Data Handling:** Stores records in nested dictionaries and allows for dynamic user additions.
*   **Automated Cleaning:** Performs text preprocessing by removing special characters (`.`, `,`, `!`, `?`), normalizing whitespace, and standardizing case.
*   **Insight Generation:** 
    *   Calculates average ratings.
    *   Identifies the longest feedback entry.
    *   Finds unique word sets and specific word frequencies (e.g., counting occurrences of "good" vs. "poor").
    *   Sorts feedback dynamically by user rating.

### 2. Python Fundamentals Assignments
*   **String Mastery:** Demonstrates indexing (forward and reverse), slicing, and method usage (`.upper()`, `.capitalize()`, `.replace()`).
*   **Data Structure Operations:**
    *   **Lists:** Creation, nested operations, sorting, and finding statistical metrics (Min, Max, Sum).
    *   **Tuples:** Demonstrates immutability, concatenation, and replication.
    *   **Sets:** Focuses on unique element storage and set math (Unions and Intersections).
    *   **Dictionaries:** Mapping student marks, updating records, and iterating through keys and values.

### 3. Utility Scripts
*   **multiplication_table.py:** A `for` loop implementation to generate tables for any user-defined number.
*   **bmi_calculator.py:** A simple tool to calculate Body Mass Index based on height and weight inputs.
*   **number_guessing_game.py:** A `while` loop logic game featuring attempt limits, randomized targets, and input validation.
